### git 从远程库克隆
	1. 登陆GitHub，创建一个新的仓库，勾选Initialize this repository with a README，这样GitHub会自动为我们创建一个README.md文件；
	2. 用命令git clone克隆一个本地库：
	
	* $ git clone git@github.com:CherrmhZ/XXX.git

PS：Git支持多种协议，包括https，但通过ssh支持的原生git协议速度最快。
